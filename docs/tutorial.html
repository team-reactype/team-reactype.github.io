Intro to ReacType

Prerequisites: Advanced knowledge in HTML, CSS, React, TypeScript, Quantum Physics, Organic Chemistry ... just kidding! Only your excitement to get started!

Sign-in page

Sign up for a ReacType account, with your Github, or just continue as a guest. Registered users enjoy additional project-saving functionality.

Start a Project (only after registration)

Users can create a new Next.js, Gatsby.js, or classic React project with just the click of a button! Registered users have the option of saving their projects to their account so they can return to them anytime.

Add Components

Users can create new components in the bottom panel. Components can be associated with a route, a link, or they can be used within other components.

Delete Components

Components can be deleted by clicking on the component with your mouse, and then pressing the delete key. Be careful when deleting components however! Upon deletion, all instances of the component will be removed within the project. The clear canvas button can also be used to start the project over from zero.

Create Custom HTML Elements

Users can create custom elements or add provided HTML elements into the application. Once the project is exported, the HTML tags generated in the code preview will function as defined in the app. Users can specify functionalities for custom elements in the code preview. The tutorial on HTML Elements explains in greater detail how this can be accomplished.

Delete Custom HTML Elements

Custom HTML elements will appear in the left sidebar, and can be deleted by clicking on the "X" button adjacent to the element. Be careful when deleting custom elements: All instances of the element will be deleted within the project.

Create and Manage Context

Create, update, and delete context that you can later assign to specific components. A tree will allow you to visualize the context hierarchy.

Create Instances on Canvas

Each component has its own canvas. Add an element to a component by dragging it onto the canvas. Div components are arbitrarily nestable and useful for complex layouts. Next.js and Gatsby.js projects have Link components to enable client-side navigation to other routes.

Component Tree

The component tree hierarchy can be viewed simply by clicking on the "Component Tree" tab in the bottom panel.

Update Styling

Select an element on the canvas to update its basic styleattributes on the bottom panel. As you create new instances and add styling, watch as your code dynamically generates in the code preview in the bottom panel and turns into a live demo.

Dark Mode

With the click of a button, users can toggle between light mode and a newly-refined dark mode!

Export Projects

Click the â€œExport" button to export the project as a full-fledged application containing TypeScripts files, Webpack, Express server, routing -- all appropriately structured depending on the project type. Clicking the "include tests" checkbox allows users to export test suites along with their project files!

Live Code Preview

Clicking the "Code Preview" tab in the bottom panel opens the code editor. Here, users can make changes to the code, such as adding styling, event handlers, elements and even components. The resulting code will be transpiled and live-rendered in the top right section.
